Art Builder App

Build stunning arts from local images with Sophon Art API.

Download


Use Art Builder App in Windows

In the download page, please download 32 bit or 64 bit windows version according to your CPU arch, if you're not sure, you can by default choose 32 bit.

After downloading, please extract the file to a folder, find an executable file named Sophon Art Builder.exe and double click to execute.

It will say "unrecognized provider", please use More Info => Run Anyway to run the app. (We're working on a way to sign the app, so we won't see this in future)

Then just fill out the form in Art Builder Application, and click on Build button to build your art files.

Use Art Builder App in Mac

In the download page, please download mac dmg file.

After downloading, double click on the file to execute, you'll see a dialog open. Drag Sophon Art Builder to Applications folder. Press F4 and you should be able to find Sophon Art Builder.

It will say "Sophon Art Builder" can't be opened because it is from an unidentified developer. Please click "OK". Then open system "Settings", find "Security & Privacy", you'll see it says "Sophon Art Builder" was blocked from opening because it is from an unidentified developer. Please "Open Anyway". (We're working on a way to sign the app, so we won't see this in future)

Then just fill out the form in Art Builder Application, and click on Build button to build your art files.
